
# Random Quote Generator

A random quote generator built with **pure HTML, CSS, and JavaScript**, focused on simplicity, performance, and a **cyberpunk / hacker terminal** aesthetic.

The project works completely **offline**, using a local data source for maximum speed and portability.

---

## Features

- One-click random quote generation
- Local data source (no external APIs)
- Cyberpunk UI with neon effects and monospace typography
- Lightweight and responsive interface
- Clean, readable, and well-structured code
- No dependencies or external libraries

---

## Technologies Used

- Semantic HTML5
- CSS3 (Flexbox, neon effects, clip-path)
- JavaScript (DOM manipulation, random logic)
- Google Fonts (Fira Code)

---

## Project Structure

``` 
├── index.html
├── style.css
└── script.js
```

## Usage

- Clone or download the repository.
- Open the `index.html` file in any modern web browser.
- Click the **“New Quote”** button to generate a random quote.
- No installation steps or configuration required.

## Technical Notes

- Quotes are stored in a local JavaScript array of objects.
- Random selection is handled using `Math.random()`.
- The DOM updates dynamically without page reloads.
- The UI prioritizes readability and contrast in dark environments.
